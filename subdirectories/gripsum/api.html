Grill Ipsum - API

Your new least favorite and most unnessecary Web API!

Despite the dedicated Grill Ipsum page being unavailable, the new experimental Gripsum API is live and well.

How to use? It's simple, dingus! Just copy and paste the following script link into your HTML document:

<script src="https://JWabbit017.github.io/subdirectories/gripsum/js/gripsum-api.js" defer></script>

Then you can apply the class attribute "gripsum", "gripsum short" or "gripsum long" to an element of your choosing, and have it filled with the third-tastiest filler text on the web!
